# midi-monitor

midi monitor chord detection plugin inspired by Chordie and MIDI Lens and Ableton MIDI Monitor m4l etcetc

operates as a silent synth device because ableton doesn't support external MIDI effect devices.
i'm sorry everybody else

## Compiling

1. install [cargo-truce](https://crates.io/crates/cargo-truce)
2. `cargo truce install --user --vst3` will build you a VST3 and install it on your system
